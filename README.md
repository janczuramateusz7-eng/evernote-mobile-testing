# 📱 Evernote Mobile Testing Project

## 📌 Project Overview
This project presents mobile testing of the Evernote application.

The main goal was to perform functional and performance testing, identify defects, and document results.

---

## 🎯 Scope of Testing
- Functional testing
- Usability testing
- Performance testing (CPU & battery)

---

## 🛠 Tools Used
- Jira
- Mobile device / emulator
- CPU Monitor
- Screen Recorder

---

## ✅ What I Did
- Created test story in Jira
- Executed test scenarios
- Reported bugs
- Monitored CPU and battery usage
- Collected screenshots and evidence

---

## 🐞 Defects
Bug reports are available in the `/bug-reports` folder.

---

## 📊 Performance
CPU and battery usage results are in `/performance`.
