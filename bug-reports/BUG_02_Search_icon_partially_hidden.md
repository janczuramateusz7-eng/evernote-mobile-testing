# BUG 02 — Search icon partially hidden in Notes tab

## Description
The search icon is not fully visible in the search bar on the Notes tab.

---

## Steps to Reproduce
1. Open the Evernote app
2. Tap on the “Notatki” (Notes) tab at the bottom of the screen
3. Observe the search bar

---

## Expected Result
The search icon should be fully visible inside the search bar.

---

## Actual Result
The search icon is partially hidden and appears outside of its expected position.

---

## Severity
Minor

## Priority
Low

---

## Environment
- App Version: Evernote 10.17.1.1 (1245306)
- Device: Realme RMX3085
- Platform: Android
- OS Version: Android 13

---

## Notes
This is a visual UI issue that affects the layout but does not impact functionality.
