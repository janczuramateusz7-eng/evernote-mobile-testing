# Mobile Testing Checklist — Evernote App

## 📱 App Launch & Stability
- App launches without crash
- App loads within acceptable time
- No freezes during startup

---

## 🔐 Login / Access (if applicable)
- User can open the app without issues
- No unexpected login prompts
- Session works correctly

---

## 📝 Notes Functionality
- Create a new note
- Edit existing note
- Save note correctly
- Delete note
- Open note without delay

---

## 🔍 Search Functionality
- Search bar is visible
- Search input works correctly
- Suggestions are displayed
- Selecting suggestion works correctly
- Search results are relevant and readable

---

## 🧭 Navigation
- Switching between tabs works correctly
- No UI glitches during navigation
- Buttons respond properly

---

## 🎨 UI / UX
- All elements are visible
- Icons are properly aligned
- No overlapping elements
- Text is readable
- Layout is consistent

---

## ⚙️ Performance
- App runs smoothly
- No noticeable lag
- No overheating
- Acceptable CPU usage (basic observation)

---

## 📊 Device Compatibility
- App works on Android 13
- No crashes on tested device (Realme RMX3085)

---

## 🐞 Error Handling
- No unexpected crashes
- Errors (if any) are displayed properly
